FindDocuments

FindDocuments[collection]
returns all documents from the collection.


selects documents that satisfy the query.

DetailsDetails

  • The following options can be given:
  • "Limit"InfinityLimits the number of returned documents.
    "Offset"0Skips the specified number of documents.
    "Fields"AllSelects a subset of fields to return.
  • Queries may contain any of the MongoDB query operators. For a list of operators, see https://docs.mongodb.org/manual/reference/operator/query/.
